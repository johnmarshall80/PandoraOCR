 allocmit - allocates and initializes a new mit structure of specified
            length.
 appendmit - takes an mis filename and an mis entry index and appends
             them as a single string to the end of an mit structure.
 changemit - replaces the value at a specified position with an mit
             structure with a new mis filename and entry index.
 freemit - deallocates the memory associated with the specified
           mit structure.
 misfrmit - creates a new mis structure containing entries extracted
            from mis files at specified indides based on the contents
            of a mit structure.
 readmitfile - reads the contents of the specified file into a new
               mit structure.
 writemitfile - writes the contents of a mit structure to the specified
                filename.
 extractmit - returns the string containing an mis filename and entry
              index from the specified position of an mit structure.
